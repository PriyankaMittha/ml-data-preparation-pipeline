# 🚀 ML Data Preparation Pipeline

## 📌 Overview
This project demonstrates a complete Machine Learning data preprocessing workflow.  
It covers essential data cleaning and feature engineering techniques required before model building.

The goal of this repository is to build a strong foundation in preparing raw datasets for Machine Learning models.

---

## 🛠️ Key Features

✔ Handling Missing Values  
✔ Feature Engineering  
✔ Feature Extraction  
✔ Categorical Encoding  
✔ Data Scaling & Normalization  
✔ Outlier Detection & Treatment  
✔ Data Transformation  
✔ Feature Selection  

---

## 🔎 Techniques Used

### 1️⃣ Handling Missing Values
- Mean / Median Imputation
- Mode Imputation
- Forward & Backward Fill
- Dropping Null Values

### 2️⃣ Feature Engineering
- Creating new features from existing ones
- Binning & Bucketing
- Interaction Features
- Date-time feature extraction

### 3️⃣ Feature Extraction
- Text vectorization (TF-IDF / CountVectorizer)
- Extracting numerical patterns
- Dimensionality reduction (PCA)

### 4️⃣ Encoding
- Label Encoding
- One-Hot Encoding
- Target Encoding

### 5️⃣ Scaling & Normalization
- StandardScaler
- MinMaxScaler
- RobustScaler
